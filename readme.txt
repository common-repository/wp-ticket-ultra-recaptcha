This is a free add-on for WP Ticket Ultra Plugin. 

== Description ==

<a href='https://wpticketultra.com/'>WP Ticket Ultra Recaptcha</a> offers a great suite of features that will help you to offer Top-Notch Customer Support.

This plugin needs <a href='https://wordpress.org/plugins/wp-ticket-ultra/'>WP Ticket Ultra</a> and it can be used to display a reCaptcha code on multiple pages such as Registration, Login and Password Reset links. Since spam and form abuse are two critical security related issues we're offering this add-on for free.

<h4>Do you need to implement reCAPTCHA in your WordPress?</h4>

Check <a href='https://wordpress.org/plugins/ultimate-captcha/'>Ultimate Captcha</a>, this plugin allows you to add reCAPTCHA on registration, login, password reset and comments form avoiding spam attacks.

<h4>DOCUMENTATION</h4>

<a href='http://docs.wpticketultra.com/installing-recaptcha/'>CLICK HERE</a> to check the documentation.


== Installation ==

1. Upload `wp-ticket-ultra-recaptcha` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
